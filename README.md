# Fresh Milk Shop | Premium 2026 HTML Template

This is a **free, open-source static HTML template** for a modern dairy or fresh milk delivery business. It uses a high-end, luxury dark theme with gold accents, designed for great first impressions.

## Features

- **Luxury 2026 Design**: Very modern and rich layout using standard HTML and CSS.
- **Persistent Cart**: A smooth shopping cart that stays active as you browse between pages.
- **Detailed Product Pages**: Displays image, benefits, nutritional facts, and FAQs.
- **Pincode Delivery Checker**: Customers can check if you deliver to their area before they shop.
- **Responsive Layout**: Works perfectly on phones, tablets, and computers.
- **Our Farm Tour**: A special "Our Farm" page with realistic images to build trust.
- **Legal Pages**: Professional templates for Privacy Policy, Terms, Shipping, and Refunds.
- **Mobile Menu**: Smooth-glide mobile navigation.
- **No Heavy Frameworks**: Built with pure HTML5, CSS3, and Vanilla JavaScript.

## How to use this template

1. **Download or Clone**: Get all the files from this repository.
2. **Open index.html**: You can see the website by opening the `index.html` file in any browser.
3. **Edit Products**: Go to `js/products-data.js` to add your own milk or dairy products.
4. **Change Colors**: You can easily change the main colors in `style.css` under the `:root` section.
